# Project 4: Tinybit Pro Robot Control

## Project Overview

This individual project uses a micro:bit and a Tinybit Pro robot to demonstrate basic robot control with MicroPython.

The robot is programmed to perform movement, directional control, speed control, and predefined path control.

## Main Functions

* Forward movement
* Backward movement
* Left spin
* Right spin
* Automatic stop
* Low-speed control
* Medium-speed control
* High-speed control
* Predefined square path control

## Hardware Components

* BBC micro:bit V2
* Yahboom Tinybit Pro robot chassis
* Two DC motors
* Rechargeable battery
* Ultrasonic sensor
* Motor control board

## Software and Tools

* MicroPython
* Mu Editor
* Tinybit MicroPython library
* GitHub

## Source Code Files

* `forward_movement.py`
* `backward_movement.py`
* `spin_left.py`
* `spin_right.py`
* `low_speed.py`
* `medium_speed.py`
* `high_speed.py`
* `square_path.py`

## Testing Results

The robot was tested successfully for forward movement, backward movement, left spin, right spin, three speed levels, and predefined square path control.

## Development Process

The project was developed incrementally. Each function was tested separately and uploaded to GitHub with a meaningful commit message.
